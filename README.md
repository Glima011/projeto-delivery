# Matsu Sushi

Projeto de cardápio digital para restaurante japonês.

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Objetivo
Criar uma plataforma simples e moderna para apresentação do cardápio e realização de pedidos via WhatsApp.
